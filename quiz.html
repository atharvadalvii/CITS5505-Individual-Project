Web basics quiz

There are 15 multiple-choice questions loaded with AJAX from data/questions.json (they are not embedded in this HTML file). Topics include HTML, CSS, and JavaScript fundamentals from the tutorial: semantic elements, CSS variables/layout decisions, DOM events, safe text updates, and browser storage basics. The order is randomised on every page load using a Fisher–Yates shuffle.

Pass mark: 70% (inclusive). Your score, percentage, and pass/fail status appear below after you submit. If you pass, a bonus quote is fetched from a public API (never hard-coded in the source).

After you select your first answer, the page treats the quiz as in progress: you get a native beforeunload prompt if you try to refresh, close the tab, or follow a link to another page. Switching to another tab does not show that prompt (browsers do not allow it), but when you return you may see a short reminder here. The prompt and reminder stop after a successful submission.
